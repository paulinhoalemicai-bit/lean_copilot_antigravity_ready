# Lean Copilot MVP

## Como rodar

1. Crie o ambiente virtual:

```bash
python -m venv .venv
```

2. Ative o ambiente:

### Windows PowerShell

```powershell
.venv\Scripts\Activate.ps1
```

3. Instale as dependências:

```bash
python -m pip install -r requirements.txt
```

4. Copie `.env.example` para `.env` e coloque sua chave.

5. Rode:

```bash
python -m streamlit run app.py
```
