# Versão para Antigravity

Esta versão foi organizada com os arquivos principais na raiz do projeto para facilitar importação.

## Arquivos principais
- app.py
- coach.py
- config.py
- db.py
- requirements.txt
- .env.example

## Passos após importar
1. Renomeie `.env.example` para `.env`
2. Preencha `OPENAI_API_KEY`
3. Instale dependências de `requirements.txt`
4. Rode `python -m streamlit run app.py`

## Observações
- Banco local: `leancopilot.db`
- Entrada principal: `app.py`
